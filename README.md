## Getting started
Publish the files in the 'dist' folder on your website.
OR
run on a local server using Python and ngrok.

```bash
  $> cd /path/to/your-project-folder/dist
  $> python -m http.server 8080
  ```

``` bash
  $> cd /path/to/your-project-folder/dist
  $> ngrok http 8080
  ```
